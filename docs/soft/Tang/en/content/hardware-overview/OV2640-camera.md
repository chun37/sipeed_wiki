## Features

+ 2 Megapixel
+ Array size: UXGA 1600X1200
+ Power supply: 3.3V
+ IO voltage level: 1.7V~3.3V DC
+ Output formats:
    - YUV(422/420)/YCnCr422
    - RGB565/555
    - 8-bit compressed data
+ Max image transfer rate:
    - UXGA/SXGA 15fps
    - UXGA/SXGA 30fps
    - SVGA 30fps
    - CIF 60fps
